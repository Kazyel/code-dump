Grid

This is a some few hours project with the goal to put in practice the skills acquired in this course taught on Frontend Masters.

  • Basic Templating
  • Span & Areas

Basic Templating

So, the first lesson is about basic templating with grid. We can use grid-template-columns and grid-template-rows to define our colums and rows, respectively.

Span & Areas

The second lesson is mostly about good usage of spans and grid areas. We can use grid-column and grid-row to define the area and disposition of our grid items.

Overlapping Areas

The third lesson is about manipulating grid-column and grid-row to create overlapping areas.

Subgrid

The fourth lesson is about manipulating grid-column and grid-row to create subgrids.
